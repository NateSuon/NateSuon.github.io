USDA Forest Service

Supported the modernization of a 40+ application platform as Product Designer - gathering requirements and developing design artifacts in order to improve the agency's UX maturity by building reusable components/patterns within a new design system. This initiative developed web and mobile applications that manage and track the natural resources across all United States National Forests.

Mobile Field Collection

Under tight constraints of a 1-year deadline - we delivered a product that satisfies all functionality of the legacy application with improved design that meets standard Android Material system patterns and Section 508 Compliance.

We put together a project plan and developed a cadence where we were able to gather user research and produce design specifications for development teams to implement within the following sprint.

Application Features Data Entry, Interactive & Responsive Tables, Data Collection, User Accessibility in Harsh Environmental Conditions
Design Artifacts Heuristic Analysis, Information Architecture, Service Blueprint, Interactive Prototypes, Complete Design Specifications

User Testing

When some design concepts received mixed feedback and difficult communication, we proposed conducting user testing to help support implementation of decisions. Over the next sprint, we were able to develop several options for an interactive prototype and conduct user testing sessions with field users to gather valuable data.

Design Hand-off

After receiving approval from PO on design decisions, we provided design specifications to the development team for implementation.
